# 翔葵丸サイト更新 - ユーザー要件と意思決定記録

## セッション日時
2025年1月4日

## ユーザーからの初期要求

### 1. ブランディング変更
**要求**: "イカ釣りデビューしませんか？とかイカ釣り専門船ではなくなったので、同じような箇所があれば、修正してください。"

**背景**: 
- 翔葵丸は元々イカ釣り専門船として運営
- 現在はSLJ（スーパーライトジギング）、ナイトティップラン、イカメタルの3種類の釣り体験を提供
- ブランディングを「イカ専門」から「総合釣り船」へ転換

**実施内容**:
- 8箇所の"イカ釣り"表記を"釣り"に変更
- SEOメタデータ全面更新
- ガイドページを総合釣りガイドに書き換え

### 2. レンタルタックルの季節制限明記
**要求**: "レンタルタックルがあるのは、夏のイカの時期だけなので、勘違いしないようなUIにしてください。"

**理由**: 
- レンタルタックルは6月〜9月のケンサキイカシーズンのみ利用可能
- 顧客が通年利用できると誤解する可能性

**実施内容**:
- `pricing/page.tsx`に赤文字で季節制限を明記
- "※レンタルタックルは6月〜9月のケンサキイカシーズン（イカメタル・オモリグ）のみご利用いただけます。"

### 3. ラベルデザイン統一
**要求**: "ナイトティップランのタイトル？てきなやつは、スーパーライトジギング（SLJ）とおなじようなラベルにしてください。"

**詳細**: 
- SLJとナイトティップランのラベルグラデーションを統一
- 従来: ナイトティップラン `from-accent-600 to-primary-600`（オレンジ→青）
- 変更後: `from-primary-600 to-secondary-600`（青→明るい青）

### 4. ケンサキイカプラン追加
**要求**: "料金表からケンサキイカが消えています。"

**ユーザー意図の確認**:
- 質問: 「ナイトティップランに説明追加 vs 独立したイカメタルプラン作成？」
- 回答: **「イカメタル・オモリグで！」** → 新規プラン作成を選択

**詳細仕様の確認**:
- レンタルタックル利用期間: 「6月〜9月（ケンサキイカシーズン）」
- ラベル色: 「SLJと完全に同じ青系グラデーション」
- 料金: 「同じ（9,000円/45,000円）」

### 5. アナウンスバナー文字色変更
**要求**: 
```
🎊 NEW スーパーライトジギング・ナイトティップラン 受付開始しました！
萩沖の豊かな漁場で、多彩な魚種との出会いをお楽しみください
の文字色は、#1d242f？かな？これにして！
```

**理由**: グラデーション背景上での視認性向上

**実施内容**: 
- `page.tsx`の3箇所で`text-[#1d242f]`に変更
- 白文字から濃いグレーへ変更でコントラスト改善

## ユーザー意思決定の詳細

### イカメタル・オモリグプラン仕様
**確定事項**:
1. プラン名: 「イカメタル・オモリグ」
2. 対象魚: ケンサキイカ（アカイカ）
3. シーズン: 6月〜9月
4. 料金:
   - 乗り合い: ¥9,000
   - チャーター: ¥45,000
5. レンタルタックル: 利用可能（+¥1,000）
6. ラベルグラデーション: `from-primary-600 to-secondary-600`

### ガイドページ方針
**選択肢**:
1. イカ釣りガイドのまま（ケンサキ・スルメの詳細充実）
2. 総合釣りガイドに書き換え（SLJ・ティップラン・イカメタル全て）

**ユーザー決定**: **「総合釣りガイドに書き換え」**

**理由（推測）**:
- ブランド方針と一貫性（イカ専門→総合釣り船）
- 初心者向けに全釣法を網羅的に案内
- SEO効果（多様なキーワード対応）

## 実装上の技術的決定

### Structured Data戦略
**決定**: 6つのOfferを定義
- SLJ 乗合・チャーター
- ナイトティップラン 乗合・チャーター
- イカメタル・オモリグ 乗合・チャーター

**理由**:
- 各釣法・プラン種別を個別に検索エンジンに伝達
- validFrom/validThroughで季節限定プランを明示
- Rich Results表示の可能性向上

### ガイドページ構成
**最終構成**:
1. 3つの釣法紹介カード（概要）
2. 各釣法の詳細ガイド（SLJ → ティップラン → イカメタル）
3. 共通アドバイス4カード（持ち物、船酔い、安全、マナー）
4. 初心者応援セクション（統計数字）
5. CTA（予約・FAQ）

**設計思想**:
- 初心者が迷わない情報構造
- 各釣法の特徴を平等に紹介
- レンタルタックルをイカメタルセクションで強調

## 将来的な検討事項

### 1. 季節プラン表示の動的制御
**現状**: 6月〜9月限定プランも通年表示
**検討**: シーズン外は非表示またはグレーアウト

### 2. レンタルタックル拡充
**現状**: イカメタルのみ
**可能性**: SLJ・ティップランにも拡大検討時は`pricing/page.tsx`更新

### 3. 釣果データ連携
**アイデア**: 
- 実際の釣果から「狙える魚」更新
- 季節ごとの実績カラー情報を反映
- ガイドページに季節別おすすめ情報追加

### 4. 予約システム統合
**現状**: お問い合わせフォームのみ
**検討**: カレンダー予約システム導入の可能性
